UR10 逆动力学求解器 — 技术文档

基于 ROS2 Humble pluginlib 的 UR10 串联机械臂逆动力学求解器，含 MATLAB 系统辨识模型、Sigmoid 摩擦补偿与负载动力学回归矩阵

1. 项目概述

本项目是一个 ROS2 (Humble Hawksbill) 软件包集合，用于求解 UR10 六轴工业机器人的逆动力学问题。由意大利萨莱诺大学 (Università degli Studi di Salerno) 自动控制组的 Vincenzo Petrone、Enrico Ferrentino 和 Antonio Annunziata 开发，基于 BSD 3-Clause 许可证开源。

核心功能：给定机器人关节位置 $q$、速度 $\dot{q}$ 和加速度 $\ddot{q}$，计算维持该运动所需的：

  • 关节力矩 $\tau$：总驱动力矩（包含惯性 + 科里奥利 + 重力 + 摩擦 + 负载）
  • 惯性矩阵 $H(q) \in \mathbb{R}^{6\times6}$：关节空间的对称正定质量矩阵
  • 科里奥利向量 $C(q,\dot{q})\dot{q}$：速度相关的非线性力
  • 重力向量 $g(q)$：仅与位形相关的重力补偿
  • 摩擦力向量 $f(\dot{q})$：基于 Sigmoid 非线性模型的关节摩擦

1.2 核心技术特点

  • 电流级建模：在电机电流层面（而非力矩层面）建立回归模型。力矩通过实测电机增益矩阵 $K = \text{diag}(K_1,\dots,K_6)$ 转换：$\tau = K \cdot i$
  • 双寄存器架构：机械臂本体动力学（电流级）和负载动力学（力矩级）分离，负载部分通过 $Y_L(q,\dot{q},\ddot{q}) \in \mathbb{R}^{6\times19}$ 的线性回归矩阵支持任意末端工具的运行时补偿
  • Sigmoid 非线性摩擦：$i_{\text{friction},j} = f_{v,j}\dot{q}_j + f_{o,j} + \frac{f_{c,j}}{1 + e^{-\alpha_j(\dot{q}_j + n_j)}}$，每个关节 5 个参数，消除了经典库仑摩擦在零速处的不连续性
  • MATLAB 自动代码生成：8 个 C++ 函数（合计约 25,000 行）由 Symbolic Math Toolbox 符号推导 → MATLAB Coder 5.6 自动生成，保证了辨识模型到运行时代码的零误差传递

2. 文档模块

以下四份技术文档覆盖了从运动学到辨识的完整知识链。正运动学和逆运动学的实现位于上游 MATLAB 仿真平台（同仓库的 test/virtual/ 和 realtime_trajectory_fit_gui/），逆动力学是本项目的核心，电流模型辨识则连接了两侧代码库。

正运动学

给定 6 个关节角度 q，通过 URDF 模型计算末端 TCP 的齐次变换矩阵（位姿）。

输入: q ∈ ℝ⁶ (rad) → 输出: T ∈ SE(3)
上游 MATLAB URDF

逆运动学

给定末端 TCP 目标位姿，通过四阶段渐进松弛数值 IK 求解器计算关节角度。

输入: T ∈ SE(3) → 输出: q ∈ ℝ⁶ (rad)
上游 MATLAB 四阶段松弛

逆动力学 ★ 核心

给定 q, dq, ddq，通过电流级辨识模型计算惯性 + 科里奥利 + 重力 + Sigmoid 摩擦的总力矩。

输入: q,dq,ddq ∈ ℝ⁶ → 输出: τ ∈ ℝ⁶ (Nm)
本项目的核心 8个生成函数 Sigmoid摩擦

电流模型辨识

基于实测 CSV 数据，通过 Simulink 管道验证线性电流估计模型，输出逐关节 RMSE/MAE/MaxAbs。

输入: 实测数据 → 输出: 参数 + 误差指标
上游 MATLAB Simulink

gtest / launch_testing—C++ 单元测试框架
