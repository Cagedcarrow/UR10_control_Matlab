项目概览

realtime_trajectory_fit_gui 是一个 MATLAB GUI 应用，用于 UR10 挖掘机器人的实时轨迹规划与拟合。其核心功能是：

  1. 根据料斗几何参数生成三段式挖掘轨迹（切入 → 圆弧挖掘 → 提升）
  2. 通过逆运动学 (IK) 将笛卡尔空间轨迹转换为关节空间轨迹
  3. 进行碰撞检测（自碰撞、工具-本体、工具-料斗）
  4. 在 3D 场景中可视化并播放机械臂运动

计算密集型任务（IK 求解 + 碰撞检测）通过 C++/MEX 加速，相比纯 MATLAB 实现获得 8.1 倍 加速比，单点处理时间从 217 ms 降至 27 ms。
