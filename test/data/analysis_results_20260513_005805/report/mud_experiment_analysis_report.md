## 10. 结论

### 主要发现

1. **实际实验中铲泥质量最大的组**: 05_10_175513，角度 20°、速度 35%、深度 60mm，铲泥质量 240g。
2. **响应面预测中铲泥质量最大组合**: 角度 50°、速度 100%、深度 45mm，预测质量 148g。
3. **多目标优化推荐组合**: 角度 50°、速度 100%、深度 45mm，综合评分 0.737。

### 因素影响排序

- **铲泥质量** 影响因素排序: depth_mm > angle_deg > speed_percent
- **能耗** 影响因素排序: angle_deg > speed_percent > depth_mm
- **阻力峰值** 影响因素排序: angle_deg > depth_mm > speed_percent

### 后续实验建议

1. **响应面验证**: 在多目标优化推荐参数组合附近进行重复实验验证。
2. **增加重复实验**: 当前每个参数组合只有一次实验，无法评估实验误差。
3. **控制泥浆状态**: 泥浆含水量、粘度等可能对铲泥质量有显著干扰。
4. **扩展参数范围**: 考虑更大角度（如 60°）、更高速、更深度的组合。
5. **动态力矩监测**: 考虑实时监测关节力矩，预防超载。

### 局限性说明

- 当前样本量有限（27 组），每个参数组合缺少真正重复实验。
- 泥浆状态（含水量、粘度、密度）未作为控制变量，可能对 weight_g 有干扰。
- 响应面模型基于二次多项式假设，实际物理过程可能更复杂。
- 响应面模型需要后续重复实验验证其预测能力。
- 能耗计算基于 tau_estimated 和 Act_qd，实际电气损耗未纳入。
