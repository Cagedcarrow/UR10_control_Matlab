轨迹生成算法

概述

轨迹生成由 generate_trajectory_3d 函数完成，根据料斗几何参数和铲斗姿态生成三段式挖掘轨迹：

  1. 切入段 (Approach)：铲斗从泥面以上沿斜线切入泥中
  2. 圆弧段 (Arc)：铲斗在泥中沿圆弧挖掘
  3. 提升段 (Lift)：铲斗从泥中沿斜线提升出泥

参数配置

轨迹参数通过 YAML 文件 (trajectory_params_3d.yaml) 配置：

  • basin.* — 料斗几何尺寸（内壁 X/Y/Z 范围、底面Z坐标）
  • mudHeight — 泥面高度（相对料斗底面）
  • approachAngleDeg — 切入角度（相对水平面）
  • liftAngleDeg — 提升角度
  • arcRadius — 圆弧半径
  • depthBelowMud — 泥下深度
  • clearanceAboveMud — 泥上安全高度
  • entryX / exitX — 切入/切出位置的 X 坐标

分段轨迹生成

1. 切入段 (Approach Segment)

从料斗上方安全高度斜向切入泥中：

  • 起点：泥面上方 clearanceAboveMud 处
  • 终点：泥面下方 depthBelowMud 处（圆弧起点）
  • 方向：沿 approachAngleDeg 斜向下
  • 工具朝向：X 轴对齐路径切线，Z 轴（铲斗法向）指向路径内侧

2. 圆弧段 (Arc Segment)

铲斗在泥面以下沿圆弧轨迹挖掘：

  • 圆弧从切入终点到提升起点
  • 圆弧半径 = arcRadius
  • 圆心位于泥面下方
  • 工具朝向：X 轴对齐切线，Z 轴指向圆心（保持挖掘姿态）

3. 提升段 (Lift Segment)

从泥中斜向提升出泥面：

  • 起点：圆弧终点（泥面以下）
  • 终点：泥面上方 clearanceAboveMud 处
  • 方向：沿 liftAngleDeg 斜向上
  • 工具朝向：世界 Z 轴向上（铲斗放平）

姿态定义

每个轨迹点的姿态（旋转矩阵）由路径切线方向决定：

X_axis = -tangent (铲斗前进方向 = 路径切向反方向)
Z_axis = 世界Z轴投影 (切入/圆弧段: 指向路径内侧; 提升段: 世界向上)
Y_axis = Z × X (右手定则)

然后在 MATLAB 中通过 rtfg_utils('applyPoseToTrajectory') 应用用户指定的铲斗位置 (x, y, z) 和朝向 (roll, pitch, yaw)。

目标位姿序列生成

对于 MEX 求解器，每个轨迹点需要转换为 4×4 齐次变换矩阵 (targetTforms)：

  • 位置：轨迹点的世界坐标 (x, y, z)
  • 姿态：由路径切线和段类型决定的旋转矩阵
  • 段标签：approach / arc / lift
